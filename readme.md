# Meals App

A modern and responsive meals recipe web application built with React.

## Features

*  Search meals by name
*  Browse meals by category
*  Filter by cuisine (area)
*  Add/remove favorites (stored in localStorage)
*  Fully responsive design (mobile & desktop)
*  Fast performance with Vite

## Technologies Used

* React
* React Router DOM
* SCSS (SASS)
* Context API
* Vite

##  UI/UX Highlights

* Clean and modern design
* Responsive navbar with mobile menu
* Hero section with background image
* Smooth layout and spacing

##  Project Structure

* `components/` → reusable UI components
* `pages/` → main pages (Home, Recipes, etc.)
* `context/` → global state (Favorites)
* `assets/` → images & icons
